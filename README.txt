
Het iTVonline channel
 
ondersteuning voor:
- Live TV
- Bekijken van gids en inplannen van opnames
- Programma gemist
- Trailer van films
- Beheren van opnames
 
nog een paar dingetjes,
 
- Je server moet binnen en KPN netwerk zijn om dit volledig te laten werken
- Je cliënt kan wel buiten het netwerk zijn.
- Nog niet alle zenders hebben art. (heb je een mooie afbeelding post de link hier dan voeg ik hem toe)
- Alleen ondersteuning voor players die encrypted HLS streams ondersteunen

Player die encrypted HLS streams ondersteunen (voor deze plugin):

- Plex/iOS
- Plex/Android
- PlexConnect
- Plex/Web
- Plex Samsung Smart Hub
- Plex Roku
- Plex Home Theater
Bron : https://forums.plex.tv/index.php/topic/27030-rel-nfl-network/

Installeren doe je door de repository te downloaden als zip-bestand en uit te pakken met bv winrar
de map iTVonline.bundle moet je vervolgens naar je plug-in folder kopiëren

Plugin Map
synology   : /volume1/Plex/Library/Application Support/Plex Media Server/Plug-ins
Linux      : /var/lib/plexmediaserver/Library/Application Support/Plex Media Server/Plug-ins
OSX        : ~/Library/Application Support/Plex Media Server/Plug-ins
Windows xp : C:\Documents and Settings\<GEBRUIKER>\Local Settings\Application Data\Plex Media Server\Plug-ins
Windows 7  : C:\Users\<GEBRUIKER>\AppData\Local\Plex Media Server\Plug-ins

vragen op opmerkingen zijn altijd welkom
